**Part C — Migrate .glass utility to use tokens (per D-11):**

Replace the hardcoded values in `.glass` (lines 137-142):

FROM:
```css
.glass {
  background: rgba(255, 255, 255, 0.80);
  backdrop-filter: blur(12px) saturate(160%);

  border: 1px solid rgba(0, 0, 0, 0.06);

```

TO:
```css
.glass {
  background: var(--glass-bg);
  backdrop-filter: blur(12px) saturate(160%);

  border: 1px solid var(--glass-border);
}
```

Remove the `.dark .glass` override block entirely (lines 144-147) since the tokens already handle dark mode via CSS variables.
